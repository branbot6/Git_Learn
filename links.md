# Recommended Git Learning Links

## 1) Learn Git Branching (Top Pick)
[Learn Git Branching](https://learngitbranching.js.org/)

Interactive in-browser practice. You can run Git commands and immediately see branch history changes, which makes abstract topics like branching and rebase much easier to understand.

Additional notes:
- Best starting point if you are still building intuition.
- Fully free and no setup is required.

## 2) Git by Example
[Git by Example](https://antonz.org/git-by-example/)

Scenario-based explanations that cover real workflows such as push, pull, conflict resolution, and team collaboration.

Additional notes:
- Great for filling practical gaps after you know the basics.
- Use it as a reference when you get stuck in day-to-day work.

## 3) GitHub `git_training`
[GitHub - git_training](https://github.com/UnseenWizzard/git_training)

A hands-on training repository that walks you through merge conflicts and manual conflict resolution in a local environment.

Additional notes:
- Good for practicing real conflict cases step by step.
- Clone the repo and complete the exercises locally.
